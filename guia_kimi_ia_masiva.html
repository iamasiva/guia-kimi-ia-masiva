Una ruta directa para entender las funciones clave de Kimi y usarlas como sistema de trabajo: crear, investigar, programar, automatizar y convertir conocimiento en procesos.

La idea central

Kimi no es solo un chat. Es una caja de herramientas para transformar trabajo pesado en entregables listos.

Primero entiende esto

La mayoria usa la IA para preguntar. Kimi apunta a algo mas potente: usar IA para producir. Presentaciones, sitios web, documentos, investigaciones, hojas de calculo, codigo, agentes y habilidades reutilizables. Si sabes que boton usar, ahorras horas.

01 / Slides

Crea presentaciones profesionales con estructura y diseño

Convierte ideas, documentos largos o reportes en diapositivas claras, ordenadas y listas para presentar.

Cuando usarlo

Si tienes un informe pesado y necesitas explicarlo en 10, 15 o 20 slides sin empezar desde una pantalla en blanco.

Prompt util Convierte este informe en una presentacion ejecutiva para inversionistas. Quiero narrativa clara, datos clave, riesgos, oportunidades y conclusion accionable.
02 / Websites

Pasa de una idea a un sitio web funcional sin programar desde cero

Describe la pagina que quieres, muestra una referencia visual o parte de un mockup, y Kimi puede ayudarte a generar una version web lista para probar.

Cuando usarlo

Para prototipos, landing pages, paginas de producto, MVPs o primeras versiones de una idea digital.

Prompt util Crea una landing page responsive para este producto. Incluye beneficios, precios, testimonios, formulario de registro y CTA principal.
03 / Docs

Analiza, resume y mejora documentos largos sin perderte en paginas

Trabaja con PDFs, Word, Markdown y documentos tecnicos para extraer lo importante, revisar contenido y convertirlo en entregables mas utiles.

Cuando usarlo

Contratos, propuestas, informes, manuales, papers, politicas internas o cualquier documento que tome demasiado tiempo leer completo.

Prompt util Revisa estos documentos y crea una tabla con puntos clave, riesgos, diferencias, inconsistencias y recomendaciones concretas.
04 / Deep Research

Haz investigaciones profundas sin abrir veinte pestañas

Deep Research busca, compara, sintetiza y organiza informacion compleja en un reporte con criterio.

Cuando usarlo

Para analizar empresas, mercados, tendencias, competidores, tecnologias, inversiones o temas donde una respuesta corta no alcanza.

Prompt util Haz una investigacion profunda sobre esta empresa. Analiza mercado, competidores, riesgos, modelo de negocio, proyecciones y oportunidades.
05 / Sheets

Crea hojas de calculo con formulas, tablas y modelos listos

Genera modelos financieros, dashboards, tablas, CSV, formulas y graficos sin construir todo manualmente.

Cuando usarlo

Cuando necesitas ordenar datos, crear un modelo DCF, hacer analisis de sensibilidad o producir una hoja que otra persona pueda usar.

Prompt util Crea un modelo DCF con supuestos, flujo de caja proyectado, WACC, valor terminal, sensibilidad y graficos editables.
06 / Agent Swarm

Divide tareas enormes entre muchos agentes trabajando en paralelo

Agent Swarm sirve para tareas masivas donde un solo agente se queda corto o tardaria demasiado.

Cuando usarlo

Analizar muchas empresas, revisar grandes cantidades de informacion, crear reportes extensos o comparar mercados completos.

Prompt util Divide esta investigacion en subtareas: perfiles, datos financieros, competidores, tendencias, riesgos y reporte ejecutivo final.
07 / Kimi Code

Haz que la IA entienda tu repositorio y trabaje con codigo real

Kimi Code puede analizar proyectos, ejecutar comandos, refactorizar, corregir errores y ayudar en tareas tecnicas de desarrollo.

Cuando usarlo

Para migraciones, bugs, refactors, pruebas, lectura de codigo legacy o automatizacion de tareas repetitivas de programacion.

Prompt util Analiza este repositorio, identifica los archivos relevantes, aplica la migracion solicitada y ejecuta las pruebas hasta que todo pase.
08 / Kimi Claw

Automatiza tareas recurrentes con un agente que sigue trabajando

Kimi Claw apunta a tareas persistentes: monitorear, investigar, reportar y repetir procesos con continuidad.

Cuando usarlo

Para reportes diarios, seguimiento de mercado, monitoreo de noticias, alertas, revision de competidores o flujos programados.

Prompt util Cada dia a las 7:00 a.m., analiza Bitcoin: precio, noticias, niveles clave, tendencia y escenarios posibles. Entrega resumen accionable.
Estrategia IA MASIVA

No uses todas las funciones. Usa la funcion correcta para el cuello de botella correcto.

Si el problema es explicar, usa Slides. Si el problema es investigar, usa Deep Research. Si el problema es repetir un proceso, usa Presets o Skills. Si el problema es escalar una tarea, usa Agent Swarm o Claw.

01

Identifica que parte del trabajo te roba mas tiempo.

02

Elige la funcion de Kimi que convierte ese trabajo en entregable.

03

Guarda el proceso como preset o skill si lo vas a repetir.

09 / Skills

Convierte procesos, documentos y prompts en habilidades reutilizables

Skills es la zona donde Kimi deja de ser solo un chat y empieza a comportarse como un sistema de trabajo. Aqui puedes usar habilidades ya creadas, crear nuevas hablando con Kimi, convertir documentos en habilidades y gestionar las que ya tienes.

Que incluye esta seccion

Habilidades: utilidades listas para tareas especificas como investigar, crear documentos, trabajar con PDFs o manipular hojas de calculo. Document to Skill: convierte archivos de Office o documentos internos en una habilidad reutilizable. Create with Kimi: permite crear una habilidad conversando con Kimi. Manage Skills: sirve para administrar las habilidades disponibles.

Cuando usarlo

Cuando tienes un workflow que vas a repetir muchas veces: una metodologia de analisis, una plantilla de contenido, un proceso de investigacion, una guia interna o un documento que quieres transformar en accion.

Prompt util Convierte este documento en una skill reutilizable. Debe aplicar la metodologia paso a paso, pedir los datos necesarios y entregar un resultado estandarizado.
